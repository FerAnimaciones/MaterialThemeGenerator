Icon Caveats

The icon treatment is not supported yet via icon fonts. Currently the Material icon font only supports the filled treatment. This project has produced SVG Icon sets that can be used with the MatIconRegistry. These fonts were derived from the Material UI project. 

      
constructor(
  matIconRegistry: MatIconRegistry,
  domSanitizer: DomSanitizer
) {
  matIconRegistry.addSvgIconSet(
    domSanitizer.bypassSecurityTrustResourceUrl(
      `/assets/material-icons-Outlined.svg`
    )
  );
}
